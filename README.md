# WASD Range

Простая 3D стрелялка в браузере: большая арена, плавное WASD-движение, спринт, raycast-выстрелы, здоровье, рекорд по убитым зомби и волны ЕГЭ-зомби.

## Запуск без Node.js

Дважды кликни:

```text
start-game.cmd
```

Или запусти из PowerShell:

```powershell
.\start-game.cmd
```

Скрипт сам откроет браузер на `http://127.0.0.1:4173/`.

## Управление

- WASD - движение
- Мышь - обзор
- ЛКМ - выстрел
- Shift - бег
- V или кнопка в правом верхнем углу - переключить вид
- 1-5 - выбрать или купить оружие
- Esc - отпустить курсор

## Цель

Переживай волны зомби. Они бегут к игроку и наносят урон рядом. За убийства и зачистку волн дают деньги, на них можно покупать лук, автомат, меч и пулемет. После зачистки волны здоровье немного восстанавливается.

Зомби используют картинку `download.png` из корня проекта. Если ее нет, игра пробует `assets/zombie-cover.png`, потом встроенную запасную обложку.

## Запуск через Node.js

```powershell
node server.mjs
```

После запуска открой:

```text
http://127.0.0.1:4173/
```

Игра использует Three.js с CDN, поэтому браузеру нужен доступ к `unpkg.com`.
